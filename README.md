# CNC刀具参数助手

在线工具：https://panxiangbin.github.io/cnc-parameter-demo

## 功能介绍

根据材材、硬度、刀具、加工方式等条件，给出推荐的线速、转速、进给等参数范围。

包含功能：
- 材料硬度推荐参数
- 6种常用计算器
- 10种常见异常排查
- 新增：切深(ap)、切宽(ae)调整
- 线程底孔计算器
- 结果导出、历史记录
- 深色模式

## 使用方式

1. 选择材料、硬度、加工方式、刀具类型
2. 输入刀具直径、刀具刃数
3. 填写机床刚性和装夹情况
4. 点击“生成建议”

## 注意事项

本工具只作参考，实际加工请小余量试切，并综合刀具牌号、冷却、装夹等因素判断。
